属性名称 属性类型 是否必须 说明
name String 是 安卓应用的名称
md5 String 是 安卓安装包文件的MD5值
package String 是 安卓应用发布的报名
version String 是 安卓应用的版本号
type Int(0~2) 是 安卓应用的类型，有训练样本、测试样本、记录样本，对应0、1、2
maxSDK String 否 最大适用SDK版本
minSDK String 否 最小适用SDK版本
targetSDK String 否 目标SDK版本
signName String 否 签名名称
certIssuer String 否 签名者信息
rootTrusted Boolean 否 根证书是否受信任
selfSigned Boolean 否 是否自签名
verifySign Boolean 否 签名是否受信任
subjectDN String 否 证书颁发机构的信息
mainActivity String 否 安卓应用的主界面活动
modifyDate String 是 最后更改的日期时间，用户可以不设置，系统会自动添加
publicationDate String 否 安卓应用发行的日期时间
level Int(0-3) 是 安卓应用的风险等级，0级代表正常，越高说明风险越大
desc String 否 对安卓应用的描述，比如与某已知恶意病毒接近
hrefList List(URL) 否 一些与该应用有关的外部链接
